# InvoiceCraft

A feature-rich Invoice Management System built with **Next.js 14**, **Tailwind CSS**, and **Google Sheets** as the backend.

## Features
- **Dashboard**: Real-time analytics (Revenue, VAT, Paid/Unpaid Stats).
- **Invoice Editor**: Create, Edit, and PDF Generation (A4 format).
- **Google Sheets Sync**: Zero-database setup; all data is stored in your Google Sheet.
- **Role-Based Auth**: Admin & User roles with secure login.
- **Multi-Theme**: Customizable UI colors.
- **History & Filtering**: Track payments and overdue invoices.

## Quick Start Only

1.  **Install**: `npm install`
2.  **Configure**: Set up `.env.local` (See `DOCUMENTATION.md` or `.env.example`).
3.  **Run**: `npm run dev`

For detailed setup instructions, database schema, and API reference, please see **[DOCUMENTATION.md](./DOCUMENTATION.md)**.
